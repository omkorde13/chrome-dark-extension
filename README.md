# 🌙 Chrome Dark Mode Extension

A lightweight and efficient browser extension that applies a sleek dark theme to websites, reducing eye strain and improving readability.

## 🚀 Features
- **Instant Dark Mode:** Toggle dark themes on any website with one click.
- **Performance Focused:** Minimal impact on browser memory and page load times.
- **Smart Inversion:** Intelligently handles images and videos to prevent color distortion.

## 🛠️ Installation
1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select this project folder.

## 💻 Tech Stack
- **JavaScript (ES6+)**: Logic for DOM manipulation and theme toggling.
- **CSS3**: Custom styles and high-contrast dark filters.
- **Chrome API**: For extension lifecycle management.
